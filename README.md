# Диверсификация портфеля на основе MIS и SMIS

Данный проект является реализацией курсовой работы на тему **«Значимые выводы при построении диверсифицированного портфеля»**. В нём сравниваются два подхода к отбору активов для диверсифицированного портфеля:

- **Классический MIS (Maximum Independent Set)** – строится пороговый граф по фиксированному коэффициенту корреляции, затем находится максимальное независимое множество.
- **SMIS (Significant MIS)** – пороговый граф строится с использованием множественной проверки гипотез (процедуры Бенджамини–Хохберга или Холма), что позволяет статистически обоснованно удалять шумовые связи.

Портфели оптимизируются методом минимальной дисперсии (Марковиц), после чего сравниваются по доходности, риску, коэффициенту Шарпа, эффективному числу ставок (ENB) и индексу Херфиндаля–Хиршмана (HHI).

## Установка и запуск

1. **Клонируйте репозиторий**:
 ```bash
 git clone https://github.com/your-username/MIS_SMIS_Comparsion.git
 cd MIS_SMIS_Comparsion
```
2. **Создайте виртуальное окружение**:
```bash
  python -m venv venv
  source venv/bin/activate   # Linux/Mac
  venv\Scripts\activate      # Windows
```
4. **Установите зависимости**:
```bash
  pip install -r requirements.txt
```
6. **Запустите эксперимент**:
```bash
 python run.py --market NASDAQ --smis_procedure BH --mis_p 0.5 --smis_p 0.1 --viz True
```
 Доступные аргументы:

--market – NASDAQ или MOEX (по умолчанию NASDAQ)

--smis_procedure – BH (Бенджамини–Хохберг) или Holm (по умолчанию BH)

--mis_p – порог корреляции для T-графа (по умолчанию 0.5)

--smis_p – уровень значимости для S-графа (по умолчанию 0.1)

--top_n – число акций с наибольшим коэффициентом Шарпа (по умолчанию 100)

--viz – флаг (без значения) для включения визуализации

## Описание работы
1. **Загрузка данных**
Для NASDAQ используется библиотека yfinance (топ-200 акций по капитализации),
для MOEX – moexalgo (все акции). Данные загружаются за период с 15.04.2023 по 15.04.2025.

3. **Предобработка**
Рассчитываются логарифмические доходности, удаляются пропуски и бесконечные значения. Затем отбираются top_n акций с наибольшим коэффициентом Шарпа.

4. **Построение графа**

T-граф – ребро между активами добавляется, если |корреляция| > mis_p.

S-граф – рёбра определяются по результатам процедуры Бенджамини–Хохберга или Холма с уровнем smis_p.

5. **Поиск MIS**
Максимальное независимое множество находится как максимальная клика в дополнении графа (точный алгоритм Брона–Кербоша). Результат сохраняется в data/mis_result/.

6. **Оптимизация портфеля**
Для найденных активов решается задача минимизации дисперсии с ограничением на сумму весов = 1 и запретом коротких позиций (используется cvxpy).

7. **Сравнение портфелей**
Вычисляются годовая доходность, риск, коэффициент Шарпа, ENB и HHI. Проводятся парный t-тест и бутстрап-тест для разности Шарпа.

- Визуализация (опционально)

- Кумулятивная доходность

- Эффективные границы

- Распределение доходностей

- Диаграммы весов

## Результаты
**Основные выводы по работе**:

Портфели на основе T-графа и S-графа с процедурой Холма дают близкие результаты (статистически неразличимые).

Процедура Бенджамини–Хохберга приводит к более плотному графу, MIS получается меньше, портфель более концентрированный (выше риск, но потенциально выше доходность).

Эвристический подход с порогом 0.4–0.5 остаётся надёжным и простым в использовании.
